# Lab 9 - Sort (Part 2)
Complete the exercises in `Sort.h` (class/function definitions - no `main()` function should be written).

In this lab you will implement a few Sort algorithm tasks.

## Run command
`make test`

In order to test functions individually, you can use the following:
`make test try={functionName}`

For example, to try a function `bubblePass()` function, you can run:
`make test try=bubblePass` *(No spaces)*

You may also try:
`make test try=bubbleSort`
`make test try=insertionStep`
`make test try=insertionSort`
`make test try=swap`
`make test try=minFind`
`make test try=newSort`

### Demo code

You can also try your own demo code to test your functions in `demo.cpp`. To run this code, you can build and run the file, or you can simply run the command `make demo` and the project will be built and run for you.
